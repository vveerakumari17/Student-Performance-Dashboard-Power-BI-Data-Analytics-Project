# 📊 Student Performance Dashboard | Power BI

## 📌 Project Overview
This project presents an interactive Power BI dashboard for analyzing student performance. It helps identify patterns in academic scores and understand how different factors such as gender, parental education, lunch type, and test preparation affect student performance.

---

## 🎯 Objectives
- Analyze student performance across multiple subjects.
- Compare scores by gender.
- Evaluate the impact of parental education.
- Analyze pass/fail rates.
- Identify performance trends using interactive visualizations.

---

## 📂 Dataset
The dataset contains over **1,000 student records** with the following columns:

- Gender
- Race/Ethnicity
- Parental Level of Education
- Lunch Type
- Test Preparation Course
- Math Score
- Reading Score
- Writing Score
- Average Score
- Pass/Fail Status

---

## 📊 Dashboard Features

- 📌 KPI Cards
  - Total Students
  - Average Score
  - Pass Count
  - Fail Count

- 📈 Subject-wise Score Analysis

- 👨‍🎓 Gender-wise Performance

- 🎓 Parent Education Analysis

- 🍽️ Lunch Type Comparison

- 📚 Test Preparation Impact

- ✅ Pass vs Fail Analysis

- 🎛️ Interactive Slicers

---

## 🛠 Tools Used

- Power BI
- Microsoft Excel
- Power Query
- DAX

---

## 📈 Key Insights

- Compare student performance across different subjects.
- Identify the effect of parental education on academic scores.
- Evaluate the impact of completing the test preparation course.
- Analyze overall pass and fail percentages.
- Discover trends through interactive filters.
